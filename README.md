# asynchronous-FIFO
Asynchronous FIFO with 64 positions and using 6 bits for data.
I have built this FIFO memory to enhance my skills in SystemVerilog, synchronous and asynchronous concepts, testbenches logic and to learn new things.
It is nothing fancy, I am an electrical and computer student in my third year of college and I am justing building stuff to learn new concepts, SystemVerilog/Verilog, testbenches skills and to other things that college doesn't teach.
I have written a "FAQ" type file where I explored my doubts while I was building my FIFO, so if you are like me, who is here to learn something, if you are reading my code and if you don't understand something then I recommend you to visit this file because your answer can probably be there.
